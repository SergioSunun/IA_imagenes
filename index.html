¿Qué es un Prompt?

Un Prompt es la instrucción que le damos a una IA para que genere un resultado (texto, imagen, código, etc.). Un prompt efectivo se compone de varios elementos clave que guían a la IA hacia el resultado deseado.

1. Propósito/Sujeto (El Qué)

Define el objeto o idea principal de la generación. Es el núcleo de tu petición.

Ej: Un auto volador, un debate sobre ética, un personaje de fantasía.

2. Estilo/Técnica (El Cómo)

Define la estética o la forma de ejecución. ¿Quieres una foto, una pintura, un poema?

Ej: Estilo Cyberpunk, como una pintura al óleo, tono formal, en verso.

3. Contexto/Detalles (El Dónde)

Añade detalles que ambientan y refinan el resultado, creando una escena completa.

Ej: En el año 2077, bajo una lluvia ácida, con un fondo minimalista.

4. Parámetros (Opcional)

Restricciones técnicas o preferencias específicas que la IA debe seguir.

Ej: Tonos azules y morados, relación de aspecto 16:9, sin texto.

5. Público (Opcional)

Define a quién va dirigido el contenido, especialmente útil para la generación de texto.

Ej: Dirigido a estudiantes, para profesionales de marketing.

Laboratorio de Prompts

Existen cuatro formas fundamentales de interactuar con la IA para generar contenido. Explora cada una para entender su propósito y ver cómo se construyen. Selecciona un tipo de prompt para comenzar.

Producir: Creación con un Objetivo

Se utiliza para crear una imagen o texto con un propósito específico y único. Es el tipo de prompt más directo: pides exactamente lo que necesitas para un fin concreto, como un anuncio, un ícono o una ilustración para un artículo.

"Un estudiante con laptop usando IA, estilo futurista, tonos azules y neón, sentado en un aula virtual con ventanas a una ciudad voladora. Detalle de iluminación cinematográfica, fotografía de alta resolución."

Explorar: Búsqueda de Variaciones

Ideal para la fase de lluvia de ideas. Partes de un concepto central y pides a la IA que genere múltiples variaciones artísticas, cambiando estilos, elementos o paletas de colores para encontrar la mejor dirección creativa.

"Un cerebro con polvos de colores en fondo negro, estilo surrealista, variaciones en neón y metálico."

Transformar: Cambio de Estilo Radical

Consiste en aplicar un estilo artístico o un medio completamente diferente a un sujeto o escena. Es como "traducir" una idea de un lenguaje visual a otro, por ejemplo, convertir una foto en un personaje de anime.

Concepto Base:

Un programador frente a pantallas.

+

Estilo Aplicado:

Estilo Cyberpunk con luces de neón.

Prompt Resultante:

"Un programador frente a pantallas, transformado a estilo cyberpunk con luces de neón, lluvia ácida, implantes biónicos y moda techwear."

Describir: Ingeniería Inversa Visual

Esta técnica consiste en analizar una imagen existente para generar el prompt que podría haberla creado. Es una excelente manera de aprender qué palabras clave (keywords) funcionan para lograr ciertos estilos y efectos.

Análisis de una imagen de interfaz de IA:

  • Sujeto Principal: Predicción de textos o interfaz de machine learning.
  • Elementos Visuales: Pantallas múltiples, líneas luminosas, palabras flotando.
  • Paleta: Azul eléctrico, púrpura, acentos magenta.
  • Tono: Tecnológico, limpio, con profundidad de campo y efecto bokeh.

¡Ahora es tu Turno!

La mejor forma de aprender es practicando. Aquí tienes algunas actividades para afianzar tus nuevas habilidades en la creación de prompts. ¡Desarrolla tu maestría!

Actividad 1: Producción

Escribe un prompt para promocionar tu colegio en redes sociales. Incluye un propósito claro (ej. "atraer nuevos estudiantes"), un estilo visual (ej. "vibrante y moderno") y una paleta de colores específica.

Actividad 2: Exploración

En parejas, redacten un prompt artístico sobre un tema libre (ej. "la amistad en el futuro"). Incluyan un estilo base y al menos 2 variaciones que cambien radicalmente la estética.

Tarea: Transformación

Redacta un prompt de transformación que tome un objeto o personaje famoso y lo cambie a un estilo radicalmente opuesto. Por ejemplo: convertir a "Harry Potter" en un personaje al estilo de Studio Ghibli, o "La Monalisa" en una escultura low-poly con neón.
